# рџЊџ Project Overview

Atelier Jewellery is a luxury jewellery e-commerce application that combines a modern customer shopping experience with a powerful administrator dashboard.

Customers can browse premium jewellery collections, search products, manage their cart and wishlist, receive AI-powered jewellery suggestions, and place demo orders.

Administrators can manage products, inventory, pricing, categories, featured collections, and customer-facing content through a dedicated dashboard.

The application is built with a responsive and scalable architecture suitable for future integration with real payment gateways and cloud databases.
